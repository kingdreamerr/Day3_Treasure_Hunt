## How to run this on your device

- Clone this repository
```
git clone https://github.com/kingdreamerr/Day3_Treasure_Hunt.git
```
- cd into the repo
```
cd Day3_Treasure_Hunt
```

- Paste the following in the terminal 
```
python3 main.py
```
